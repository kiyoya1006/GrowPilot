## GrowPilotとは

GrowPilot AI Japanは、中小企業・個人事業主向けに

- AI活用支援
- Web制作
- SEO対策
- マーケティング
- 業務自動化

を通して、お客様の事業成長を伴走支援するブランドです。

ホームページを制作することが目的ではありません。

**「見つけられる → 信頼される → 問い合わせにつながる」**

この流れを一緒に育てることを使命としています。
